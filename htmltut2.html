HTML Elements (continued)

Elements

<section>

The HTML <section> element represents a generic section of a document or application. It can be used to group together related content.

<section>
    <h2>Section Title</h2>
    <p>Section content...</p>
</section>

<nav>

The HTML <nav> element represents a section of a page that links to other pages or to parts within the page: a section with navigation links.

<nav>
    <a href="#home">Home</a>
    <a href="#about">About</a>
    <a href="#contact">Contact</a>
</nav>

<article>

The HTML <article> element represents a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable.

<article>
    <h2>Article Title</h2>
    <p>Article content...</p>
</article>

<aside>

The HTML <aside> element represents a portion of a document whose content is only indirectly related to the document's main content.

<aside>
    <h3>Related Content</h3>
    <p>Aside content...</p>
</aside>

<footer>

The HTML <footer> element represents a footer for its nearest sectioning content or sectioning root element. A footer typically contains information about the author, copyright data, or links to related documents.

<footer>
    <p>Copyright Â© 2024 Example Corp.</p>
</footer>

<details> and <summary>

The HTML <details> element creates a disclosure widget in which information is visible only when the widget is toggled into an "open" state. The <summary> element provides a summary or label for the details disclosure widget.

<details>
    <summary>Click to reveal details</summary>
    <p>Hidden details...</p>
</details>

<figcaption> and <figure>

The HTML <figcaption> element represents a caption or a legend associated with a figure or an illustration represented by the <figure> element.

<figure>
    <img src="image.jpg" alt="Description">
    <figcaption>Figure caption...</figcaption>
</figure>

<main>

The HTML <main> element represents the main content of the <body> of a document or application. The main content area consists of content that is directly related to or expands upon the central topic of the document or central functionality of the application.

<main>
    <h1>Main Content</h1>
    <p>Main content...</p>
</main>

<mark>

The HTML <mark> element represents text which is marked or highlighted for reference or notation purposes, due to the marked passage's relevance or importance in the enclosing context.

<p>This is a <mark>highlighted</mark> text.</p>

<time>

The HTML <time> element represents either a time on a 24-hour clock or a precise date in the Gregorian calendar, optionally with a time and a time-zone offset.

<time datetime="2024-04-07T12:00:00Z">April 7, 2024 12:00 UTC</time>
