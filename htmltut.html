<h1> - <h6>

The HTML heading elements represent six levels of section headings. <h1> is the highest level and <h6> is the lowest. Headings are defined with decreasing significance (from <h1> to <h6>).

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

<p>

The HTML <p> element represents a paragraph. Paragraphs are usually represented in visual media as blocks of text that are separated from adjacent blocks by vertical blank space and/or first-line indentation.

This is an example paragraph. It demonstrates the use of the <p> element in HTML for creating paragraphs of text.

<p>This is an example paragraph.</p>

<a>

The HTML <a> element (or anchor element), with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.

For example, this is a link to example.com.

<a href="https://www.example.com">this is a link to example.com</a>

<img>

The HTML <img> element embeds an image into the document. It is a replaced element.

For example, 

<img src="https://via.placeholder.com/150" alt="Placeholder Image">

<ul> and <li>

The HTML <ul> element represents an unordered list of items, typically rendered as a bulleted list. Each item in the list is represented by a <li> element.

  • Item 1
  • Item 2
  • Item 3
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

<ol> and <li>

The HTML <ol> element represents an ordered list of items, typically rendered as a numbered list. Each item in the list is represented by a <li> element.

  1. Item 1
  2. Item 2
  3. Item 3
<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>

<form>

The HTML <form> element represents a document section containing interactive controls for submitting information to a web server.






<form>
    <label for="name">Name:</label><br>
    <input type="text" id="name" name="name"><br>
    <label for="email">Email:</label><br>
    <input type="email" id="email" name="email"><br><br>
    <input type="submit" value="Submit">
</form>

<input>

The HTML <input> element is used to create interactive controls for web-based forms in order to accept data from the user.

Example:


<input type="text" placeholder="Enter your name"><br>
<input type="email" placeholder="Enter your email"><br>
<input type="submit" value="Submit">

<iframe>

The HTML <iframe> element represents a nested browsing context, embedding another HTML page into the current one.

<iframe src="https://www.example.com" height="200" width="300" title="Example"></iframe>

<header>

The HTML <header> element represents introductory content, typically a group of introductory or navigational aids.

Website Header

Home | About | Contact 
<header>
    <h1>Website Header</h1>
    <nav>
        <a href="#">Home</a> |
        <a href="#">About</a> |
        <a href="#">Contact</a>
    </nav>
</header>

Character Entities

Character entities are used to display reserved characters in HTML. Here are some examples:

  • &lt; - less than sign (<)
  • &gt; - greater than sign (>)
  • &amp; - ampersand (&)
  • &quot; - double quotation mark (")
  • &copy; - copyright symbol (©)
  • &reg; - registered trademark symbol (®)
<ul>
    <li>&lt; - less than sign (<)</li>
    <li>&gt; - greater than sign (>)</li>
    <li>&amp; - ampersand (&)</li>
    <li>&quot; - double quotation mark (")</li>
    <li>&copy; - copyright symbol (&copy;)</li>
    <li>&reg; - registered trademark symbol (&reg;)</li>
</ul>
